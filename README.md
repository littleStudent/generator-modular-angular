##generator-modular-angular

This is a yeoman generator for angularJS applications.

The aim of this generator is the project structure which looks something like this:

* app.js
* app.less
* ```bower_components/```
* bower.json
* ```features/```
  * ```userManagement/```
    * ```registration/```
      * controller.js
      * dataModel.js
      * service.js
      * style.less
      * view.html
    * ```login/```
    * ```settings/```
* gulpfile.js
* index.html
* ```node_modules/```
* package.json
* routes.js
